# Sistema de Agenda Telefonica com Arvore Binaria de Busca

Projeto em Java para gerenciamento de contatos utilizando uma Arvore Binaria de Busca implementada manualmente.

## Tema

Tema 05 - Sistema de Agenda Telefonica.

## Estrutura do contato

- Nome
- Telefone
- E-mail
- Cidade

## Funcionalidades

- Inserir contato
- Buscar contato pelo nome
- Remover contato
- Exibir contatos em ordem alfabetica
- Exibir contatos em pre-ordem
- Exibir contatos em pos-ordem
- Mostrar primeiro contato alfabeticamente
- Mostrar ultimo contato alfabeticamente
- Mostrar quantidade de contatos
- Mostrar altura da arvore
- Encerrar sistema

## Como executar

Compile:

```bash
javac -d bin src/principal/*.java
```

Execute:

```bash
java -cp bin principal.Principal
```
